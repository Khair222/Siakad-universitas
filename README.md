# SIAKAD - Sistem Informasi Akademik Terpadu
**D3 Teknik Informatika Vokasi USU**

Website ini adalah simulasi Sistem Informasi Akademik (SIAKAD) yang dirancang dengan antarmuka modern (**Glassmorphism**) dan fungsionalitas lengkap untuk 4 peran pengguna utama. Proyek ini dibangun menggunakan HTML, CSS, dan JavaScript murni tanpa framework (Vanilla JS).

### 2. Modern User Interface
*   **Glassmorphism**: Desain kekinian dengan efek blur, transparansi, dan gradasi warna yang segar.
*   **Responsive**: Layout yang rapi dan mudah diakses di berbagai ukuran layar.
*   **Interaktif**: Notifikasi, Modal Pop-up, dan transisi halaman yang halus.

### 3. Simulasi Sistem (Client-Side)
*   **Local Storage**: Menggunakan penyimpanan browser untuk simulasi sesi login dan data sementara (seperti status KRS).
*   **No Backend Required**: Dapat berjalan langsung tanpa setup server/database (cukup buka HTML).
